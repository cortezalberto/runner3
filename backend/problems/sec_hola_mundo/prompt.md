# Problema: Hola Mundo

Crea un programa en Python que imprima en pantalla el mensaje: "Hola Mundo!".

## Entrada
Ninguna. Este programa no lee datos del usuario.

## Salida
El programa debe imprimir exactamente: `Hola Mundo!`

## Ejemplos
```
Salida: Hola Mundo!
```

## Restricciones
- Debes implementar la función `main()` que use `print()` para mostrar el resultado
- El formato debe ser exactamente: "Hola Mundo!" (con mayúsculas en H y M)
- No modifiques la estructura de la función `main()`
