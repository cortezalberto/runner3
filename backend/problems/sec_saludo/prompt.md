# Problema: Saludo personalizado

Escribe un programa en Python que pida al usuario su nombre y luego muestre un saludo personalizado en pantalla.

## Entrada
Un string que representa el nombre del usuario.

## Salida
Un saludo con el formato: `Hola {nombre}!`

## Ejemplos
```
Entrada: Juan
Salida: Hola Juan!
```

```
Entrada: María
Salida: Hola María!
```

## Restricciones
- Debes implementar la función `main()` que lea la entrada con `input()` y use `print()` para mostrar el resultado
- El formato debe ser exactamente: "Hola {nombre}!" (con coma después de Hola y signo de exclamación al final)
- No modifiques la estructura de la función `main()`
